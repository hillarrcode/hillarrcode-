## Running Locally

1. Clone the repository:
```bash
git clone https://github.com/YOUR_USERNAME/YOUR_REPOSITORY.git
```

2. Open the `FPP` folder.

3. Launch `index.html` in your browser.

## Deploying with GitHub Pages

1. Push this repository to GitHub.
2. Go to **Settings → Pages**.
3. Under **Build and deployment**, choose **Deploy from a branch**.
4. Select the branch (`main`) and the folder containing `index.html` (or move the website files to the repository root if preferred).
5. Save and wait a few minutes for your site to be published.
